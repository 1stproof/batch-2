# Type III Factors and Index Theory

## Definitions
- **Definition 2.2.** The crossed product Mf = M ⋊ ϕ R is the von Neumann algebra acting on the Hilbert space K = H ⊗ L2 (R) = L2 (R, H) generated by the operators πσϕ (x) (x ∈ M) and λ(t) (t ∈ R) defined by (πσϕ (x)ξ)(s) = σ−tϕ (x)ξ(s), (λ(t)ξ)(s) = ξ(s − t).
- **Definition 2.3.** (Connes-Takesaki, [8]) The above-mentioned pair (Z(M), f θ | s Z(M̃) ) is called the (smooth) flow of weights of M.
- **Definition 3.4.** Ind E = E −1 (I).
- **Definition 6.10.** ([1, 60]) An automorphism θ ∈ Aut(M; N ) is called strongly outer if the following condition is satisfied for each k: we must have x = 0 (6.3) whenever x ∈ Mk satisfies nx = xθ(n) for each n ∈ N . An action θ : G −→ Aut(M; N ) is called strongly outer if each automorphism θg (g 6= e) is strongly outer.
