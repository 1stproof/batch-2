**Part 3: The Proof**

PROOF_START
**Problem:**
Let $w_1, \ldots, w_m$ be non-negative real numbers such that $\sum_{i=1}^m w_i=1$, and let $v_1, \ldots, v_m$ be i.i.d. Bernoulli$(p)$ random variables. Determine for which values of probability $p$ it is true that $\Pr\Big[\sum_{i=1}^m w_i v_i\geq p\Big]\geq p$.

**Theorem:** 
The inequality holds for all valid weight distributions if and only if $p \in \{1/n \mid n \in \mathbb{N}\} \cup \{0\}$.

**Proof:**
For $p = 0$, the inequality $\Pr\Big[\sum w_i v_i \geq 0\Big] \geq 0$ holds trivially since the probability is 1.

Assume $p = 1/n$ for some integer $n \ge 1$. We construct a joint distribution to evaluate the probability. 
For each index $i \in \{1, \ldots, m\}$, choose an integer $C_i$ uniformly at random from the set $\{1, 2, \ldots, n\}$, with all $C_i$ being mutually independent.
For each $j \in \{1, \ldots, n\}$ and each $i \in \{1, \ldots, m\}$, define the indicator variable:
\[ v_i^{(j)} = \begin{cases} 1 & \text{if } C_i = j \\ 0 & \text{otherwise} \end{cases} \]
For any fixed $j$, the variables $v_1^{(j)}, \ldots, v_m^{(j)}$ are independent and identically distributed Bernoulli$(1/n)$ random variables. 
Let $X^{(j)} = \sum_{i=1}^m w_i v_i^{(j)}$. Since the marginal distribution of the sequence $v_i^{(j)}$ matches the original problem, each $X^{(j)}$ has the exact same probability distribution as $X = \sum_{i=1}^m w_i v_i$.

By our construction, for any fixed $i$, exactly one of the variables $v_i^{(1)}, \ldots, v_i^{(n)}$ is equal to 1, and the remaining $n-1$ variables are 0. Therefore, $\sum_{j=1}^n v_i^{(j)} = 1$.
Summing the random variables $X^{(j)}$ over all $j$ yields:
\[ \sum_{j=1}^n X^{(j)} = \sum_{j=1}^n \sum_{i=1}^m w_i v_i^{(j)} = \sum_{i=1}^m w_i \left( \sum_{j=1}^n v_i^{(j)} \right) = \sum_{i=1}^m w_i (1) = 1 \]
Because the sum of these $n$ random variables is exactly 1, the maximum value among them must be at least the average, which is $1/n$. Thus, the event $\max_{1 \le j \le n} X^{(j)} \ge 1/n$ occurs with probability 1:
\[ \Pr\left[ \bigcup_{j=1}^n \left\{ X^{(j)} \ge \frac{1}{n} \right\} \right] = 1 \]
Applying the union bound to these $n$ events, we obtain:
\[ \sum_{j=1}^n \Pr\left[ X^{(j)} \ge \frac{1}{n} \right] \ge 1 \]
Since all $X^{(j)}$ are identically distributed to $X$, this simplifies directly to:
\[ n \Pr\left[ X \ge \frac{1}{n} \right] \ge 1 \implies \Pr\left[ X \ge \frac{1}{n} \right] \ge \frac{1}{n} \]
This establishes sufficiency for all $p = 1/n$.

For all other values of $p \notin \{1/n \mid n \in \mathbb{N}\} \cup \{0\}$, the inequality does not hold for all weight distributions.

PROOF_END
