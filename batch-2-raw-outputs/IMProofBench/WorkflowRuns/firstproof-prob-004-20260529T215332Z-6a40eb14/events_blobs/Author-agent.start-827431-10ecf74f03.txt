## Main conclusion

I did **not** find the mixed-face estimate in `answer.tex` as a theorem in Guth's thesis, the arXiv rectangle paper, or the minimax/cup-square paper. The closest results are in Guth thesis Chapter 8, especially Proposition 8.1.1 and Proposition 8.1.3, but they give different mixed inequalities and are stated for 2-contracting diffeomorphisms. They do not imply the desired theorem in the high-aspect range.

So the current status seems to be: the mixed estimate in `answer.tex` is a plausible new input, not a known Guth estimate. I found no actual counterexample map, but the standard Guth estimates plus the extra thesis estimates all allow the scaling counterexample below. Thus the proof still needs a genuinely new anisotropic/mixed-skeleton tightening argument, or the statement must be weakened.

## Exact Guth statements found

**Area-expanding embeddings of rectangles, arXiv:0710.0403.** The main theorem on p.1 says that if there is a \(k\)-expanding embedding \(S\hookrightarrow R\), then for all \(0\le j<k\le l\le n\),
\[
(R_1\cdots R_j)^{(l-j)/(k-j)}R_{j+1}\cdots R_l
\ge c(n)(S_1\cdots S_j)^{(l-j)/(k-j)}S_{j+1}\cdots S_l.
\]
The map version is **Estimate 1**, p.10: if \(U\subset R\) and \(\Phi:(U,\partial U)\to(S,\partial S)\) has degree \(D>0\), then
\[
\operatorname{dil}_k(\Phi)\ge c(n)Q_1\cdots Q_j
(Q_{j+1}\cdots Q_l)^{(k-j)/(l-j)},\qquad Q_i=S_i/R_i.
\]
This is explicitly not restricted to diffeomorphisms. The tightening construction is described on pp.5-7 and carried out in Section 6, pp.13-15. It tightens a complex of cycles using the rectangle isoperimetric profile; I found no mixed \(x_1x_3/x_1x_4\)-face refinement there.

**Guth thesis, Chapter 8, pp.140-149.** This is directly about “Partial Results On 2-Contracting Diffeomorphisms Between 4-Dimensional Rectangles.” On p.140, Guth lists the embedding inequalities for a 2-contracting diffeomorphism:
\[
R_1R_2\gtrsim S_1S_2,\quad
R_1R_2R_3\gtrsim S_1S_2S_3,\quad
R_1^2R_2R_3\gtrsim S_1^2S_2S_3,
\]
\[
R_1R_2R_3R_4\gtrsim S_1S_2S_3S_4,\quad
R_1^3R_2R_3R_4\gtrsim S_1^3S_2S_3S_4.
\]
These are exactly the \(n=4,k=2\) monomial estimates.

The closest result to the desired \(S_2^{1/2}S_3^{3/2}\) exponent is **Proposition 8.1.1**, p.140, for ellipsoid boundaries:
\[
B1.\ R_2R_3\gtrsim S_2S_3,\qquad
B2.\ R_2^2R_3R_4\gtrsim S_2^2S_3S_4,
\]
\[
B3a.\ \text{If }R_2^2>S_2S_3,\text{ then }R_2R_3R_4\gtrsim S_2^{1/2}S_3^{3/2}S_4,
\]
\[
B3b.\ \text{If }R_2^2<S_2S_3,\text{ then }R_3R_4\gtrsim S_3S_4.
\]
For the unit cube this gives Guth’s Theorem 8.1 condition \(S_2^{1/2}S_3^{3/2}S_4\lesssim1\) (equivalently \(S_2S_3^3S_4^2\lesssim1\)).

Also important is **Proposition 8.1.3**, p.144:
\[
\text{If } C R_1R_2^2 < S_1S_2S_3,\text{ then }R_3R_4\gtrsim S_3S_4.
\]
Under failure of the first alternative this only gives
\[
R_1R_2^2\gtrsim S_1S_2S_3,
\]
which is not enough to force
\[
R_1R_2\gtrsim S_1S_2^{1/2}S_3^{1/2}.
\]

On thesis p.130, rational-homotopy methods give additional 4D \(k=2\) inequalities:
\[
R_2R_3^2R_4\gtrsim S_2S_3^2S_4,\quad
R_1R_2R_3^2R_4\gtrsim S_1S_2S_3^2S_4,
\]
\[
R_1R_2^2R_3^2R_4\gtrsim S_1S_2^2S_3^2S_4.
\]
Guth says the full list from this method is a “combinatorial mess,” and later on p.149 says: “the techniques in this thesis are not even close to estimating the best 2-dilation of diffeomorphisms between 4-dimensional rectangles.”

**Minimax/cup-square paper.** Appendix 3, pp.53-54, gives the general minimax principle:
\[
\Lambda\,\mathbb V_{(M,g)}(Oa(k,M))\ge \mathbb V_{(N,h)}(Oa(k,N))
\]
for degree-one maps and natural mod-2 operations \(O\), including cup powers and Steenrod squares. On p.54 Guth conjectures a rectangle minimax formula for \(Sq_{n-l}^Q a(k,R)\) which would imply the standard monomial estimate above. I found no mixed-tower rectangle formula that would imply the desired \(S_1S_2^{1/2}S_3^{3/2}S_4\) bound.

## Algebraic obstruction

The standard and thesis estimates do not rule out the model
\[
S=(1,1,\varepsilon^{-6},\varepsilon^{-6}),\qquad
R=(\varepsilon,\varepsilon^{-4},\varepsilon^{-5},\varepsilon^{-6}).
\]
Then
\[
R_1/S_1=\varepsilon,\qquad
R_3R_4/(S_3S_4)=\varepsilon,\qquad
\frac{\operatorname{Vol}R}{S_1S_2^{1/2}S_3^{3/2}S_4}=\varepsilon.
\]
Thus this would violate the desired theorem for any fixed \(\kappa\) if an actual 2-contracting degree-one map existed. But all known lower bounds checked are satisfied:

| estimate ratio | asymptotic value |
|---|---:|
| \(R_1R_2/(S_1S_2)\) | \(\varepsilon^{-3}\) |
| \(R_1R_2R_3/(S_1S_2S_3)\) | \(\varepsilon^{-2}\) |
| \(R_1^2R_2R_3/(S_1^2S_2S_3)\) | \(\varepsilon^{-1}\) |
| \(\operatorname{Vol}R/\operatorname{Vol}S\) | \(\varepsilon^{-2}\) |
| \(R_1^2\operatorname{Vol}R/(S_1^2\operatorname{Vol}S)\) | \(1\) |
| \(3\)-dilation estimate \(j=1,l=4\) | \(\varepsilon^{-3/2}\) |
| \(3\)-dilation estimate \(j=2,l=4\) | \(\varepsilon^{-5}\) |
| Prop. 8.1.3 ratio \(R_1R_2^2/(S_1S_2S_3)\) | \(\varepsilon^{-1}\) |
| thesis p.130 boundary rational-homotopy ratio | \(\varepsilon^{-2}\) |
| thesis p.130 double ratios | \(\varepsilon^{-1},\varepsilon^{-5}\) |

This table is reproducible with `python3 code/check_scaling_example.py`.

## Derivation attempts

From Estimate 1 and \(\operatorname{Dil}_3\le1\), the best direct volume lower bound is
\[
R_1^2\operatorname{Vol}R\gtrsim S_1^2\operatorname{Vol}S,
\]
hence under \(R_1\le\kappa S_1\),
\[
\frac{\operatorname{Vol}R}{S_1S_2^{1/2}S_3^{3/2}S_4}
\gtrsim \kappa^{-2}(S_2/S_3)^{1/2}.
\]
This proves the desired volume alternative only when \(S_3/S_2\lesssim \kappa^{-6}\).

I also checked the natural preimage/subrectangle route using the width-volume inequality for subsets of rectangles. Partitioning \(S\) into boxes and applying the \(k=2,n=4\) subset estimate reproduces the same monomial bounds; it does not generate the missing factor \((S_3/S_2)^{1/2}\). Applying Prop. 8.1.3 to the preimages of the \(S_1S_2\)-planes gives a good \(R_3R_4\) alternative when \(R_1R_2^2\) is small, but in the high-aspect model \(R_1R_2^2/(S_1S_2S_3)=\varepsilon^{-1}\), so the proposition is silent.

## Recommendation

Do not cite the mixed estimate in `answer.tex` as known. A safe next draft should state it explicitly as a new lemma/proposition still to be proved. The proof would need to refine Guth’s complex-of-cycles tightening so that the cubical \(2\)-skeleton records mixed \(x_1x_3\) and \(x_1x_4\) faces, not only the standard monomial skeleton volumes. The literature search found no existing cup-square/Steenrod-square or tightening theorem that supplies that refinement.
