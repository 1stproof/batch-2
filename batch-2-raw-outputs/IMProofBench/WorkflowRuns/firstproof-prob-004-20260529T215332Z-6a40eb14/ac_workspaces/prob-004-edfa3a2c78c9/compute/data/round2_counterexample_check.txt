One strict-inequality perturbation
  Fix constants 0 < alpha < 1 and mu > 1.
  Step 2 use a = alpha L^(-13/2), b = L^12.
  Then a^2 b = alpha^2 L^-1 < L^-1 = R2 R3 R4.
  Insert the bounded linear map
      (a,a,a,L^12) -> (a, mu a, a, L^12/mu).
  Step 3 uses A = 1, now with R2/R1 = mu > 1.
  The final target is
      S' = (alpha L^(-13/2), mu alpha L^(-13/2), 1, alpha mu^-1 L^(11/2)).
  After scaling D0 by lambda = C0^(1/2), the three ratios are
      (lambda/alpha) L^(-1/2),
      (lambda^2 mu/alpha) L^(-7/2),
      lambda^4 alpha^(-5/2) mu^(1/2) L^(-3/4).
  Hence all still tend to zero as L -> infinity.
