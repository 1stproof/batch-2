## 3. Mathematical audit

### Problem interpretation

The interpretation of â€śirreducibleâ€ť as â€śnot a nontrivial graph join,â€ť equivalently connected complement graph, is reasonable and is explicitly recorded. This satisfies the ambiguity-handling requirement.

### DKEP notation and normal-bidual setup

The definition of \(\widetilde S_X(N)\) is compatible with DKEP Lemma 8.5, but the text says that for a separable finite factor relative proper proximality is equivalent to absence of an \(N\)-central state whose restriction to \(N\) is **normal**. DKEP Lemma 8.5 states the restriction is exactly the trace; for a factor, an \(N\)-central normal state is a scalar multiple of the trace, so this is repairable, but the argument should explicitly include normalization.

### Lemma â€śordinary support bookkeepingâ€ť

This is a central technical lemma, and it is not adequately justified.

The author claims that for a hereditary \(Y\subset A_N\),
\[
