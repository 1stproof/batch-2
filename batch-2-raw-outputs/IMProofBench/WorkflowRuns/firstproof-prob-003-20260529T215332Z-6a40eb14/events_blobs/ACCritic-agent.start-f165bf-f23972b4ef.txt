## Mathematical audit

### Problem statement and interpretation

The universal interpretation is clear and faithful: determine those \(p\in[0,1]\) for which the inequality holds for every finite nonnegative probability vector. This is appropriate.

However, the same section explicitly says the sufficiency of \(0\le p\le1/3\) is unresolved. That is fatal for a complete solution.

### Positive values

The proof for \(p=0\) and \(p=1\) is correct.

The coloring proof for \(p=1/k\), \(k\ge3\), is correct. For every coloring, at least one color class has weight at least \(1/k\), and symmetry gives
\[
\Pr[W_1\ge 1/k]\ge 1/k.
\]
Color membership in a fixed color is independent Bernoulli\((1/k)\), so this proves the desired inequality.

The proof for \(p=2/b\), \(b\ge6\), rests on Lemma 1. I checked the logic of the deletion/counting proof and found it essentially valid. The last counting step should ideally spell out the even and odd cases, but I did not find a mathematical error.

The \(p=1/2\) symmetry argument is correct: \(X\) and \(1-X\) have the same distribution, so
\[
\Pr[X<1/2]=\Pr[X>1/2],
\]
which implies \(\Pr[X\ge1/2]\ge1/2\).

### Multiplicative closure

The closure proposition
\[
p,q\in\mathcal G\implies pq\in\mathcal G
\]
is correct. The product construction \(v_i=x_i y_i\) gives iid Bernoulli\((pq)\) variables, and the conditioning/normalization argument is valid.

### Base interval corollary

The corollary is correct as a conditional statement: if one proves the inequality for all \(p\in[1/6,1/3]\), then multiplicative closure with \(1/2\) gives all \(p\in[0,1/3]\).

But it remains conditional. The base interval is not proved.

### Negative examples

The exclusion of \(p\in(1/3,1/2)\) using three equal weights is correct:
\[
\Pr[\mathrm{Bin}(3,p)\ge2]=3p^2-2p^3<p.
\]

The exclusion of \(p\in(1/2,1)\) using two equal weights is also correct:
\[
\Pr[\mathrm{Bin}(2,p)=2]=p^2<p.
\]

So the negative direction is complete.

### Complement formulation

The complement formulation is correct. Since
\[
\sum_i w_i v_i = 1-\sum_i w_i(1-v_i),
\]
the event \(\sum_i w_i v_i\ge p\) is equivalent to
\[
\sum_i w_i\eta_i\le 1-p=q.
\]
Thus the desired inequality is equivalent to
\[
\Pr\!\left[\sum_i w_i\eta_i>q\right]\le q.
\]

### Pivot inequality

The pivot equivalence is valid.

For \(a=0\), the pivot inequality gives
\[
U\ge \frac{p}{1-p}(1-U),
\]
hence \(U\ge p\), recovering the small-\(p\) lemma.

Conversely, assuming the small-\(p\) lemma and adding an extra weight \(a\), conditioning on the extra Bernoulli variable gives
\[
p\,\Pr[Y\ge p-a]+(1-p)\Pr[Y\ge p]\ge p.
\]
Rearranging indeed yields
\[
\Pr[Y\ge p]\ge \frac{p}{1-p}\Pr[Y<p-a].
\]

This is a correct equivalence, but it is not a proof of the missing lemma.

### Pivot-plus-dust construction

The sharpness construction is correct. With
\[
a=p-\varepsilon,\qquad
Y_N=\frac{1-a}{N}\operatorname{Bin}(N,p),
\]
we have
\[
Y_N\to p(1-a)=p(1-p+\varepsilon)
\]
in probability, and
\[
\varepsilon<p(1-p+\varepsilon)<p.
\]
Therefore
\[
\Pr[Y_N\ge\varepsilon]\to1,\qquad
\Pr[Y_N\ge p]\to0,
\]
and the success probability tends to \(p\).

This correctly shows asymptotic sharpness. It does not establish the lower bound.

### Convex-hull formulation

The convex-hull equivalence is mostly correct.

Forward direction: valid.

Reverse direction: the overall separation argument is valid in principle, but the proof is slightly compressed. In particular, the coordinatewise down-closed property of \(P(\mathcal D)\) is asserted without proof. This can be filled by the thinning argument from the previous draft, but in the revised text it is omitted.

This omission is minor compared with the main gap, because this section is only an equivalent reformulation of the missing lemma.

### Partition-function formulation

The partition-function statement
\[
p\mathbf1\notin P(\mathcal D)
\Longrightarrow
\sum_{A\in\mathcal D}\lambda^{|A|}
\le (1+\lambda)^{m-1}
\]
with \(\lambda=p/(1-p)\) is algebraically equivalent to the convex-hull formulation. The conversion using
\[
\mu_p(\mathcal D)=(1+\lambda)^{-m}\sum_{A\in\mathcal D}\lambda^{|A|}
\]
is correct.

Again, this is not a proof.

Therefore the revised `answer.tex` is not a complete rigorous solution.
