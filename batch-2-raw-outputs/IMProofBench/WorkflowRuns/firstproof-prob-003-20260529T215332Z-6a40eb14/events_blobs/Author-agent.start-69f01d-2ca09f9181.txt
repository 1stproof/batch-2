# Round 8 compute report: small-\(p\) coalescence check

I read the current `answer.tex` and `research_notes.tex`, especially the new complement/coalescence section.  The main outcome is that the proposed pairwise coalescence assertion (C) is false, even under the nontrivial cap \(w_i\le p=1-q\).  The failure is structural, not a numerical accident.

## 1. Literature search

I did not find a published theorem that directly implies
\[
\mu_p\{x:\sum_i w_i x_i\ge p\sum_iw_i\}\ge p\qquad(p\le1/3)
\]
for arbitrary nonnegative weights.

Closest precise references checked:

- Fokkink--Meester--Pelekis, *Optimizing stakes in simultaneous bets*, ALEA 20 (2023), 153--165, DOI `10.30757/ALEA.v20-07`.  Saved as `papers/fokkink-meester-pelekis-2023-official.pdf`.
  They define
  \[
  \pi(p,t)=\sup_\gamma \Pr\Big[\sum_i c_i\beta_i\ge t\Big],
  \qquad \sum_i c_i=1,\quad \beta_i\sim\mathrm{Bernoulli}(p),
  \]
  and prove bold-play optimality in several regions.  The theorem most relevant to our complement form is Theorem 4.4:
  \[
  \frac{k}{k+1}<p\le \frac{k+1}{k+2}<t
  \quad\Longrightarrow\quad
  \pi(p,t)=p.
  \]
  Thus, for our complement target \(\Phi_q(w)=\Pr_q(S>q)\le q\), it proves the desired bound whenever the actual strict threshold \(t_*=\min\{w(A):w(A)>q\}\) lies above the next reciprocal \((k+1)/(k+2)\).  It does **not** cover the needed near-hypotenuse strip
  \[
  q<t_*\le \frac{k+1}{k+2},\qquad \frac{k}{k+1}<q<\frac{k+1}{k+2}.
  \]
  The paper's shaded-region theorem leaves exactly this kind of hypotenuse-adjacent strip; the authors also explicitly note such gaps elsewhere in the parameter diagram.

- Fishburn--Frankl--Freed--Lagarias--Odlyzko, *Probabilities for Intersecting Systems and Random Subsets of Finite Sets*, SIAM J. Algebraic Discrete Methods 7 (1986), 73--79, DOI `10.1137/0607009`.
  This gives sharp product-measure bounds for intersecting families in several cases and is used in FMP.  It gives the \(t>1/2\), \(p\le1/2\) bold-play block, but not the \(q>1/2\), \(t\downarrow q\) complement region needed here.

- Liggett-related result: the MathOverflow thread `papers/mathoverflow-278117-round8.html` records the known theorem that for \(p\ge1/2\), a convex combination \(S\) of iid Bernoulli(\(p\)) variables satisfies \(\Pr(S\ge1/2)\ge p\).  This is a fixed \(1/2\)-threshold theorem, not a through-the-mean theorem at threshold \(p\), so it does not imply the small-\(p\) lemma.

- Kellerer: I rechecked Kellerer’s *Linearkombinationen zufälliger Größen und ihre gemeinsame Verteilung*, Math. Z. 84 (1964), 403--414; EUDML metadata saved as `papers/kellerer-1964-eudml-round8.html`, existing PDF/images are in `papers/kellerer-1964-linearkombinationen-pages403-414.pdf` and `papers/kellerer_1964_pages/`.  I did not find a statement that specializes to the sharp Bernoulli halfspace inequality.  Kellerer’s “analogous regions” paper still looks related only at the convex-measure-embedding level, not as a black-box proof.

- Eaton/Hoeffding/Samuels/Feige: I downloaded Eaton’s technical report as `papers/eaton-1972-bounded-linear-combinations.pdf`.  Eaton and Hoeffding give upper-tail/normal-comparison inequalities for centered bounded sums; Samuels/Feige give small-deviation constants for sums of nonnegative variables.  These are not sharp enough for the equality regime here and do not give \(\Pr(S\ge\mathbb E S)\ge p\) with the exact constant \(p\).

## 2. Coalescence assertion is false

Recall \(p=1-q\), \(\Phi_q(w)=\Pr_q[\sum_iw_i\eta_i>q]\).  In failure variables \(F\), where each coordinate fails with probability \(p\),
\[
\Phi_q(w)=\Pr_p[w(F)<p].
\]
For a pair \(x\le y\), the exact coalescence gain is
\[
\Delta_{ij}=pq\left(
\Pr[p-x\le Z_{ij}<p]
-
\Pr[p-x-y\le Z_{ij}<p-y]
\right),
\]
where \(Z_{ij}\) is the \(p\)-biased failed weight of all other coordinates.  This endpoint convention matches the strict complement event \(w(F)<p\).

There is a clean infinite family of capped counterexamples.  Fix \(n\ge4\) and
\[
\frac1n<p<\frac1{n-1},\qquad q=1-p,\qquad w_1=\cdots=w_n=\frac1n.
\]
Then \(w_i=1/n<p\) and \(2/n>p\).  Hence \(w(F)<p\) iff \(F\) is empty or a singleton, so
\[
\Phi_q(w)=q^n+n p q^{n-1}.
\]
After coalescing any pair, the merged coordinate has weight \(2/n>p\), and only the \(n-2\) unmerged singleton failures are allowed:
\[
\Phi_q(w^{ij})=q^{n-1}+(n-2)pq^{n-2}.
\]
Therefore every pair has the same gain
\[
\Delta_{ij}
=\Phi_q(w^{ij})-\Phi_q(w)
=p q^{n-2}\bigl(n-2-(n-1)q\bigr)<0,
\]
because \(q> (n-2)/(n-1)\).  Thus no pair can be coalesced without decreasing \(\Phi_q\).

Concrete rational instance:
\[
p=\frac3{10},\quad q=\frac7{10},\quad
w=(1/4,1/4,1/4,1/4).
\]
Then
\[
\Phi_q(w)=q^4+4pq^3=\frac{6517}{10000}<\frac7{10},
\]
but for every pair
\[
\Phi_q(w^{ij})=q^3+2pq^2=\frac{637}{1000},
\qquad
\Delta_{ij}=-\frac{147}{10000}.
\]
So this is not a counterexample to the small-\(p\) lemma; it is a counterexample to the proposed coalescence proof route.

The requested averaged versions also fail on this example, since all six pair gains are equal and negative.  For \(w=(1/4)^4\), \(p=3/10\),
\[
\sum_{i<j}\Delta_{ij}=-\frac{441}{5000},\quad
\sum_{i<j}w_iw_j\Delta_{ij}=-\frac{441}{80000},
\]
\[
\sum_{i<j}(w_i+w_j)\Delta_{ij}=-\frac{441}{10000},\quad
\sum_{i<j}\min(w_i,w_j)\Delta_{ij}=-\frac{441}{20000}.
\]
Thus no nonnegative pair-weighting of the tested symmetric types can certify pair existence.

## 4. Suggested pivot

The coalescence route should not be used as stated.  A salvageable version would need a dichotomy:

1. either some pair has \(\Delta_{ij}\ge0\), allowing coalescence; or
2. the vector lies in a “reciprocal chamber trap” like the equal-weight examples above, where \(\Phi_q(w)\le q\) can be proved directly by a binomial tail.

The FMP/bold-play formulation remains the most promising literature route: it would be enough to prove the missing near-hypotenuse strip \(\pi(q,t)\le q\) for \(q\ge2/3\), \(t>q\), or only for the threshold values \(t=t_*(w)\) arising from weighted subset sums.
